Project Overview

The selected artifact is the CornerGrocer Inventory Project, originally developed in CS-210: Programming Languages. This project processes inventory data using associative containers (such as maps) to track frequencies of grocery items based on user input and sales records.

Enhancements Made

For the enhancement, I improved the search and counting algorithms to reduce time complexity, implemented exception handling for edge cases (e.g., missing files or corrupt data), and added user-friendly output formatting. I also included additional functions to sort and analyze the data more effectively, demonstrating deeper use of algorithmic concepts like sorting and hash maps.
