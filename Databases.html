Database Enhancement

This page presents my enhanced artifact in the category of Databases, part of my final CS 499 ePortfolio submission. This project demonstrates my ability to design, manage, and integrate relational or NoSQL databases in a way that supports real-world applications.

Project Overview

The selected project for this category is the Android Inventory App, originally developed in CS-360: Mobile Architecture and Programming. The purpose of this application was to manage product inventory using a persistent SQLite database to perform Create, Read, Update, and Delete (CRUD) operations.

Enhancements Made

For this portfolio, I improved the database structure, optimized SQL queries, and enhanced the data flow between the app and the underlying SQLite database. I also ensured secure handling of user input to protect against SQL injection and added user feedback for error states in database operations.
